# Robot Dog Mechanical Design

## Overview

This project presents the initial mechanical design of a simple quadruped robot (Robot Dog) created using Tinkercad.

## Project Objectives

- Design the robot body
- Design four legs
- Define the joints and degrees of freedom
- Select suitable servo motors
- Calculate the initial torque
- Consider stability and center of gravity
- Propose a walking method
- Identify possible mechanical problems

## Software Used

- Tinkercad
